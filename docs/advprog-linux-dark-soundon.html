Linux Commands We Used

These examples demonstrate basic usage for each command. Remember to replace file and directory names, as well as paths, with the actual names and locations on your system.

  1. ls: List files and directories.
    • List all files and directories in the current directory: ls
    • List files and directories with detailed information: ls -l
    • List all files and directories, including hidden ones: ls -a
  1. cd: Change directory.
    • Change to a specific directory: cd /path/to/directory
    • Change to the parent directory: cd ..
    • Change to the home directory: cd
  1. pwd: Print the current working directory.
    • Print the current working directory: pwd
  1. mkdir: Create a new directory.
    • Create a directory named "new_dir" in the current directory: mkdir new_dir
    • Create a directory with multiple levels: mkdir -p path/to/new_dir
  1. rm: Remove files and directories.
    • Remove a file: rm file.txt
    • Remove a directory and its contents (recursively): rm -r directory
    • Remove a file without prompting for confirmation: rm -f file.txt
  1. cp: Copy files and directories.
    • Copy a file to another location: cp file.txt /path/to/destination
    • Copy a directory and its contents to another location: cp -r directory /path/to/destination
  1. mv: Move or rename files and directories.
    • Move a file to another location: mv file.txt /path/to/destination
    • Rename a file: mv old_name.txt new_name.txt
  1. cat: Concatenate and display file content.
    • Display the contents of a file: cat file.txt
    • Display the contents of multiple files: cat file1.txt file2.txt
    • Concatenate files and save the output to a new file: cat file1.txt file2.txt > combined.txt
  1. gcc: C Compiler.
    • Compile a C program named "program.c" and create an executable: gcc program.c -o program
    • Compile and link multiple source files: gcc file1.c file2.c -o program
  1. code: Visual Studio Code.
    • Open a file or directory in Visual Studio Code: code file.txt or code /path/to/directory
    • Open Visual Studio Code without any specific file or directory: code
