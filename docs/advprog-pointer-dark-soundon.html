Lecture: Pointers

In C, a pointer is a variable that stores the memory address of another variable. Pointers allow you to directly manipulate memory, access data indirectly, and dynamically allocate memory.

To declare a pointer variable, you use the asterisk (*) symbol before the variable name. Here's an example:

int main() {
    int num = 42;
    int *ptr;  // Pointer declaration

    ptr = &num;  // Assigning the address of 'num' to 'ptr'

    return 0;
}

In the code above, we declare an integer variable num and a pointer variable ptr. We then assign the memory address of num to ptr using the address-of operator (&). Now ptr "points" to the memory location where num is stored.

To access the value stored at the memory location pointed to by a pointer, we use the dereference operator (*). Here's an example:

int main() {
    int num = 42;
    int *ptr;

    ptr = &num;

    printf("Value of num: %d\\n", *ptr);  // Dereferencing 'ptr' to access the value

    return 0;
}

In the code above, we use the *ptr expression to dereference the pointer and retrieve the value stored in the memory location pointed to by ptr. The output will be Value of num: 42.

Pointers are often used in C for various purposes, such as dynamic memory allocation, passing variables by reference to functions, and creating complex data structures like linked lists and trees. They provide a way to directly manipulate memory, which can be powerful but also requires careful handling to avoid errors like accessing invalid memory addresses or causing memory leaks.

Introduction to Pointers

A. Definition and Purpose

int* ptr; // declaring a pointer variable of type int

B. Memory and Address

int num = 10;
int* ptr = &num; // storing the address of num in ptr

C. Pointer Variables

int* ptr; // declaring a pointer variable
*ptr = 20; // assigning a value to the memory location pointed by ptr

D. Declaring and Initializing Pointers

int* ptr = NULL; // initializing a pointer variable to NULL

E. Dereferencing Pointers

int num = 10;
int* ptr = &num;
printf("%d", *ptr); // dereferencing the pointer to access the value stored at the memory location it points to

Pointer Arithmetic

A. Increment and Decrement

int* ptr = someArray; // assuming someArray is an array of integers
ptr++; // incrementing the pointer to point to the next element in the array

B. Pointer Arithmetic Examples

int* ptr = someArray;
int value = *(ptr + 2); // accessing the value at the third element in the array

C. Array and Pointer Relationship

int array[5] = {1, 2, 3, 4, 5};
int* ptr = array; // assigning the address of the first element of the array to ptr

Dynamic Memory Allocation

A. malloc() Function

int* ptr = (int*)malloc(sizeof(int)); // dynamically allocating memory for a single integer

B. calloc() Function

int* ptr = (int*)calloc(5, sizeof(int)); // dynamically allocating memory for an array of five integers

C. realloc() Function

ptr = (int*)realloc(ptr, 10 * sizeof(int)); // reallocating memory for a larger array size

D. free() Function

free(ptr); // releasing the dynamically allocated memory

Pointers and Functions

A. Passing Pointers as Function Arguments

void changeValue(int* ptr) {
    *ptr = 20; // modifying the value at the memory location pointed by ptr
}

B. Modifying Values through Pointers

int num = 10;
changeValue(&num); // passing the address of num to the function

C. Returning Pointers from Functions

int* createArray(int size) {
    int* arr = (int*)malloc(size * sizeof(int));
    // populate the array
    return arr; // returning the pointer to the dynamically allocated array
}

Pointers and Arrays

A. Array Name as a Pointer

int array[5] = {1, 2, 3, 4, 5};
int* ptr = array; // assigning the address of the first element of the array to ptr

B. Pointer Arithmetic with Arrays

int array[5] = {1, 2, 3, 4, 5};
int* ptr = array;
int value = *(ptr + 2); // accessing the value at the third element in the array

C. Array of Pointers

int num1 = 10, num2 = 20, num3 = 30;
int* ptrArray[3] = {&num1, &num2, &num3}; // creating an array of pointers to integers

Pointers to Pointers

A. Double Indirection

int num = 10;
int* ptr = &num;
int** ptrToPtr = &ptr; // pointer to a pointer

B. Syntax and Usage

int num = 10;
int* ptr = &num;
int** ptrToPtr = &ptr;
int value = **ptrToPtr; // accessing the value using double indirection

C. Pointer to Pointer Arithmetic

int num = 10;
int* ptr = &num;
int** ptrToPtr = &ptr;
(*ptrToPtr)++; // incrementing the pointer to the next memory location

Pointers and Strings

A. Strings as Arrays of Characters

char str[] = "Hello";
char* ptr = str; // assigning the address of the first character of the string to ptr

B. Pointers and String Manipulation

char str[] = "Hello";
char* ptr = str;
printf("%c", *ptr); // accessing and printing the first character of the string

C. String Functions and Pointers

char str1[] = "Hello";
char str2[] = "World";
char* result = strcat(str1, str2); // concatenating two strings using strcat() function
