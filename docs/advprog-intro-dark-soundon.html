進階程式設計 (Advanced Programming)

Course Description

The Advanced Programming course is designed to provide students with an in-depth understanding of programming concepts and techniques, focusing primarily on the C and C++ programming languages. This course builds upon the foundational knowledge acquired in introductory programming courses and delves into advanced programming topics and practices.

  1. C Fundamentals: This module provides a comprehensive review of the essential concepts of the C programming language. Topics covered include variables, data types, control structures, functions, and input/output operations.
  1. Multi-dimensional Arrays: Students will learn how to effectively work with multi-dimensional arrays, including two-dimensional arrays and matrices. They will explore various operations and algorithms associated with multi-dimensional arrays.
  1. Pointers: This module covers the concept of pointers, enabling direct manipulation and referencing of memory addresses. Students will learn how to utilize pointers for efficient memory management, passing arguments to functions by reference, and working with dynamic data structures.
  1. Structures: Students will delve into the concept of structures, which allow the creation of user-defined data types with multiple attributes. They will learn how to declare, access, and manipulate structure variables, as well as explore the concept of nested structures.
  1. Dynamic Memory Management: This module focuses on dynamic memory allocation and deallocation in C and C++. Students will understand dynamic memory management techniques such as malloc, calloc, realloc, and free. Additionally, topics such as memory leaks, memory fragmentation, and best practices will be covered.
  1. C++ Basics: This section introduces the fundamental concepts of the C++ programming language, including classes, objects, inheritance, polymorphism, and function overloading. Students will explore the syntax and features that differentiate C++ from C.
  1. Data Structures (Queues and Stacks) and Algorithms (Sorting and Searching): This module covers important data structures like queues and stacks, along with algorithms for sorting and searching. Students will learn how to implement these data structures and apply the algorithms to solve problems efficiently.
  1. Introduction to Object-Oriented Programming (OOP): Students will gain a solid understanding of the principles and concepts of OOP. They will learn about encapsulation, inheritance, polymorphism, and abstraction, and how to apply these concepts to create modular and reusable code.

Throughout the course, students will engage in hands-on programming exercises, projects, and problem-solving activities to reinforce their understanding of the topics covered. By the end of the course, students will have acquired the skills necessary to tackle more complex programming tasks and be well-equipped to pursue advanced programming projects in C and C++.
