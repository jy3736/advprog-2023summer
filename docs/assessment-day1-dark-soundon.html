Formative Assessment : C Programming Basics

The formative assessment problem set covers the following topics:

  1. Fundamentals of C programming
  1. Data types in C
  1. Constants in C
  1. Variables and their usage
  1. Understanding scope: global and local variables
  1. Mathematical and logical operators in C

  1. What is the purpose of a variable in programming? A. To store and manipulate data. B. To perform mathematical operations. C. To execute program instructions. D. To display output. [Answer: A]
