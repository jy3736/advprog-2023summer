Lecture: C Structures

In C programming, a struct (short for structure) is a user-defined data type that allows you to combine different data types into a single entity. It allows you to create a custom data structure that can hold multiple related variables under a single name.

Here's an example of how you can define and use a struct in C:

#include <stdio.h>

// Define a struct named "Person"
struct Person {
    char name[50];
    int age;
    float height;
};

int main() {
    // Declare a variable of type "Person"
    struct Person person1;

    // Access and assign values to the struct members
    strcpy(person1.name, "John Doe");
    person1.age = 25;
    person1.height = 1.75;

    // Access and print the struct members
    printf("Name: %s\\n", person1.name);
    printf("Age: %d\\n", person1.age);
    printf("Height: %.2f\\n", person1.height);

    return 0;
}

In this code, we define a struct named "Person" that contains three members: name, age, and height. Inside the main() function, we declare a variable person1 of type Person and access its members using the dot (.) operator.

We assign values to the struct members using the strcpy() function for the name member, and direct assignment for the age and height members. Finally, we print the values of the struct members using printf().

Structs are useful when you want to group related variables together and pass them as a single entity to functions, or when you want to define custom data types for organizing your data more effectively.

Introduction to C Struct

A. Definition of a Struct: A struct in C is a composite data type that allows you to group together different variables with different data types into a single unit. It provides a way to define your own data structure with multiple elements.

// Example of defining a struct
struct Person {
    char name[50];
    int age;
    float height;
};

B. Purpose of Structs: Structs allow you to create complex data structures that can hold multiple related pieces of data. They are useful for organizing and manipulating data in a structured manner.

C. Syntax for Declaring a Struct: To declare a struct variable, you use the struct keyword followed by the struct tag (name of the struct) and the variable name.

// Example of declaring a struct variable
struct Person person1;

Creating and Accessing Struct Members

A. Declaring Struct Members: Inside a struct, you can declare members, which are variables of different data types that define the structure's elements.

// Example of declaring struct members
struct Person {
    char name[50];
    int age;
    float height;
};

B. Initializing Structs: Struct variables can be initialized using curly braces {} with values provided for each member in the order they were declared.

// Example of initializing a struct
struct Person person1 = {"John Doe", 25, 175.5};

C. Accessing Struct Members: To access the members of a struct variable, you use the dot (.) operator followed by the member name.

// Example of accessing struct members
printf("Name: %s\\n", person1.name);
printf("Age: %d\\n", person1.age);
printf("Height: %.2f\\n", person1.height);

Nested Structs and Struct Arrays

A. Creating Nested Structs: Structs can be nested within other structs, allowing you to create hierarchical data structures.

// Example of creating nested structs
struct Address {
    char street[50];
    char city[50];
};

struct Person {
    char name[50];
    int age;
    struct Address address;
};

B. Accessing Members of Nested Structs: To access members of a nested struct, you use the dot (.) operator multiple times.

// Example of accessing members of nested structs
printf("Street: %s\\n", person1.address.street);
printf("City: %s\\n", person1.address.city);

C. Declaring Struct Arrays: You can declare arrays of structs to store multiple instances of the same struct type.

// Example of declaring a struct array
struct Person people[3];

D. Accessing Struct Arrays: You can access individual elements of a struct array using the array index notation.

// Example of accessing struct array elements
printf("Name of person at index 0: %s\\n", people[0].name);

Passing Structs to Functions

A. Passing Structs by Value: By default, when you pass a struct to a function, a copy of the struct is made, and any modifications made inside the function do not affect the original struct.

// Example of passing a struct by value
void printPerson(struct Person p) {
    printf("Name: %s\\n", p.name);
    printf("Age: %d\\n", p.age);
    printf("Height: %.2f\\n", p.height);
}

// Usage
printPerson(person1);

B. Passing Structs by Reference (Pointers): To modify the original struct inside a function, you can pass a pointer to the struct, allowing direct access and modification.

// Example of passing a struct by reference
void modifyPerson(struct Person* p) {
    p->age = 30;
}

// Usage
modifyPerson(&person1);

Manipulating Structs

A. Modifying Struct Members: You can modify struct members directly by assigning new values using the assignment operator (=).

// Example of modifying struct members
person1.age = 30;

B. Assigning Structs: You can assign one struct variable to another of the same type, copying the values of all members.

// Example of assigning structs
struct Person person2;
person2 = person1;

C. Comparing Structs: Structs can be compared using the equality operator (==), which compares each member of the structs for equality.

// Example of comparing structs
if (person1.age == person2.age) {
    printf("The ages are equal.\\n");
}

Using Typedef with Structs

A. Defining Typedefs for Structs: Typedef allows you to define an alternative name (alias) for a struct type, making it easier to declare variables of that type.

// Example of defining a typedef for a struct
typedef struct {
    char name[50];
    int age;
    float height;
} Person;

B. Benefits of Typedefs for Structs: Typedefs simplify the declaration of struct variables by eliminating the need to use the struct keyword each time.

// Example of using a typedef for a struct
Person person1;
person1.age = 25;

Common Applications of Structs

A. Creating Data Structures: Structs are commonly used to create data structures such as linked lists, stacks, queues, and trees.

// Example of a linked list node
struct Node {
    int data;
    struct Node* next;
};

B. Representing Real-World Objects: Structs can be used to represent real-world objects with multiple attributes, such as employees, students, or cars.

// Example of representing an employee
struct Employee {
    char name[50];
    int age;
    float salary;
};
